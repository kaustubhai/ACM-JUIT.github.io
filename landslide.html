Landslide Monitoring System

Category : Project Hardware

THE SYSTEM DETECTS A LANDSLIDE AND INFORMS THE CONTROL ROOM AND THE PEOPLE AROUND IT THROUGH MOBILE

The system detects a landslide and informs the control room and the people around it through mobile. This system uses Arduino, GSM Module and Ultrasonic Sensors. The sensors detect landslides and the message is sent through GSM. The entire system code was controlled by Arduino.
