Automated Attendance system

Category : Project Hardware

BASED ON OPEN CV, PYTHON WHICH USES HAAR CASCADE FOR FACIAL RECOGNITION AND SIMULTANEOUSLY MARKS THE ATTENDANCE OF THE STUDENT

A project based on Open CV, Python which uses Haar Cascade for facial recognition and simultaneously marks the attendance of the student. The system also consists of an object detection module that is used to capture the entry and exit of students in a classroom and also has the ability to count the number of students in the class periodically.
